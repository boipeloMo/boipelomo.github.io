# Boipelo Moabi Portfolio

A responsive front-end developer portfolio built with HTML, CSS, and JavaScript.

## Structure

```text
.
|-- index.html
|-- .nojekyll
`-- assets
    |-- css
    |   `-- style.css
    |-- js
    |   `-- script.js
    `-- images
        |-- quiz.svg
        |-- the-voyage-vault.png
        |-- todo.svg
        `-- validation.svg
```

## Deploy To GitHub Pages

1. Create a GitHub repository, for example `portfolio`.
2. Upload every file in this folder to the repository root.
